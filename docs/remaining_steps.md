# Remaining Steps the Student Must Complete

Because the assignment explicitly prohibits generative AI from writing the script, producing the voiceover, selecting clips, or assembling the video, these tasks must be completed by the student:

1. Select the final Freudian concept.
2. Choose the final footage clips.
3. Write the thesis and voiceover script.
4. Record the voiceover.
5. Assemble and edit the video in iMovie, CapCut, Premiere, DaVinci Resolve, or another editor.
6. Create and verify captions.
7. Upload/embed the finished video through Kaltura.

The repository includes planning documents and helper scripts to make these steps easier without replacing the student's required authorship.
